# smchat

A clojure seesaw messaging GUI.

![smchatgui](http://imgur.com/dx1yE48l.png)


## Usage

Currently possible to create an exe to run on windows.

## Features

- Connecting to a different site.
- Selecting a Name
- Clearing Messages
- Changing Prompt
- Font Color
- Theme
